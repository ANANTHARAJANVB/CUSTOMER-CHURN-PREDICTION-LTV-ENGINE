# Customer Churn Prediction & Lifetime Value Engine

## Project Overview

This project predicts customer churn and analyzes customer lifetime value using the Telco Customer Churn dataset.

## Technologies Used

* Python
* PostgreSQL
* SQLAlchemy
* Pandas
* Scikit-Learn
* Metabase

## Database

* PostgreSQL Database: telco_churn_db
* Table: telco_customers

## Dashboard Features

* Total Customers KPI
* Churned Customers KPI
* Active Customers KPI
* Churn Distribution
* Contract Type vs Churn
* Tenure vs Churn
* Monthly Charges vs Churn
* Internet Service vs Churn
* Payment Method vs Churn

## Business Insights

* Month-to-month customers show higher churn.
* Long-tenure customers are more likely to stay.
* Customer payment methods influence retention.
