# CUSTOMER-CHURN-PREDICTION-LTV-ENGINE
This project predicts customer churn and estimates Customer Lifetime Value (LTV) using customer demographics, billing, and service usage data. It helps businesses improve retention strategies and focus on high-value customers. Team: Anantharajan V B, Bhavana P, Akshay M, Mohammed Rasool K M.
